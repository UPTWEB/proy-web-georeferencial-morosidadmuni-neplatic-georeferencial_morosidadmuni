# Neplatic Desktop Windows

## Uso

1. Descargar `NeplaticDesktop-Windows.zip`.
2. Descomprimir el archivo.
3. Copiar `.env.example` como `.env` dentro de la misma carpeta del ejecutable.
4. Editar `.env` con las credenciales reales de PostgreSQL, Redis y Oracle si corresponde.
5. Ejecutar `NeplaticDesktop.exe`.

## Nota

No subas el archivo `.env` al repositorio porque contiene credenciales reales.
